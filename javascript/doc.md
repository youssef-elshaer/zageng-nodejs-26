1. Project Overview

Project Name: Simple Calculator
Description:
This project is a simple JavaScript program that performs basic arithmetic operations using variables.


---

2. Technologies Used

JavaScript



---

3. Concepts Covered

Variables

Arithmetic Operators (+, -, *, /)



---

4. Features

Addition

Subtraction

Multiplication

Division



---

5. How It Works

The program stores numbers in variables and applies arithmetic operations to display results.


---

6. Code Example

let num1 = 10;
let num2 = 5;

let sum = num1 + num2;
let subtract = num1 - num2;
let multiply = num1 * num2;
let divide = num1 / num2;

console.log(sum);
console.log(subtract);
console.log(multiply);
console.log(divide);


---

7. Output

15
5
50
2


---

8. Conclusion

This project helps beginners understand how variables and arithmetic operations work in JavaScript.
